# Jaleelo Portfolio

This is the complete portfolio source project.

## Run locally

1. Open this folder in VS Code or another editor.
2. Run `npm install` once.
3. Run `npm run dev`.
4. Open the local address Vite prints in the terminal.

The real portrait asset is stored in `assets/jaleelo-portrait-alpha.png` and is loaded by `index.html`.

## Notes

- The site uses React, React Three Fiber, Three.js, GSAP, and Framer Motion through browser imports.
- Keep the `assets` folder next to `index.html` when moving or deploying the project.
